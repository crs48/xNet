# xNet Mobile

Expo/React Native app for iOS and Android.

## Development

```bash
pnpm start         # Start Expo dev server
pnpm ios           # Run on iOS simulator
```

## Build

```bash
pnpm build         # Generate native project
```

## Features

- Local-first document storage
- Native navigation (React Navigation)
- Pull-to-refresh document list
- Minimal package footprint

## Tech Stack

- **Expo SDK 52** -- React Native framework
- **React Native 0.76** -- Mobile runtime
- **React Navigation** -- Native navigation
- `@xnet/react`, `@xnet/sdk`
